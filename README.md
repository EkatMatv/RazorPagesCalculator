Задание 1
Создайте новый проект веб-приложения ASP.NET Core,
построенный на базе шаблона «ASP.NET Core Razor Pages».
После того, как проект был создан, удалите страницу «Privacy»
(«Privacy.cshtml» и «Privacy.cshtml.cs») и все ссылки на неё из
верхнего навигационного меню.
Задание 2
На главной странице («Index») разместите форму для
пользовательского ввода.
Форма должна содержать следующие элементы:
• Поле для ввода первого числа;
• Поле для ввода второго числа;
• Элемент пользовательского интерфейса, позволяющий
выбрать арифметическую операцию (например, выпадающий
список);
• Кнопку «Вычислить».
Обратите внимание, что поля для ввода чисел являются
обязательными, а также предполагают ввод только цифр.
Задание 3
Реализуйте веб-приложение калькулятора таким образом,
чтобы после того, как пользователь ввел необходимые данные в
форму, выбрал арифметическую операцию и нажал кнопку
«Вычислить», результат вычисления отобразился на той же
(главной) странице.
Необходимо реализовать логику для следующих
арифметических операций:
• Сложение;
• Вычитание;
• Умножение;
• Деление.
